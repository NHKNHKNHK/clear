# List、Set、Map之间的区别是什么？

|            | List     | Set                           | Map                           |
| ---------- | -------- | ----------------------------- | ----------------------------- |
| 单列/双列  | 单列集合 | 单列集合                      | 双列集合，存储键值对          |
| 索引       | 支持     |                               | x                             |
| 重复       | 重复     | x                             | key不重复，value可重复        |
| 存取一致性 | 一致     | LinkedHashSet一致，其余不一致 | LinkedHashMap一致，其余不一致 |
