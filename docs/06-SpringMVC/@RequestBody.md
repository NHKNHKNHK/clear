# @RequestBody

| 名称 | @RequestBody                                                 |
| ---- | ------------------------------------------------------------ |
| 类型 | ==形参注解==                                                 |
| 位置 | SpringMVC控制器方法形参定义前面                              |
| 作用 | **将请求中请求体所包含的数据传递给控制器方法参数上**，此注解一个处理器方法只能使用一次 |
