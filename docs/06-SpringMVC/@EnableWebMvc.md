# @EnableWebMvc

| 名称 | @EnableWebMvc                                             |
| ---- | --------------------------------------------------------- |
| 类型 | ==配置类注解==                                            |
| 位置 | SpringMVC配置类定义上方                                   |
| 作用 | 开启SpringMVC多项辅助功能（包含开启json数据类型自动转换） |
